### 脚手架 创建

```sh
npx create-electron-app@latest my-new-app --template=vite-typescript
```

### Vite + TypeScript

##### 自行安装  vite 相关

```json
"devDependencies": {
    "@vitejs/plugin-react": "^4.2.1",

    "vite": "^7.3.1"
}
```
